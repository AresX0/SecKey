Enterprise Security Platform

SecKey

A modern .NET 10 platform for Intune, Entra ID, Conditional Access, Endpoint Security, and Win32 app lifecycle automation through Microsoft Graph.

Installation Guide MSI Build Notes

Core Capabilities

  • Device compliance, configuration, settings catalog, and enrollment profiles
  • Conditional Access and named locations management
  • Entra groups, users, app registration, and PIM scheduling operations
  • Win32 LOB packaging and staged Azure blob upload pipeline
  • WPF UI + CLI for operators and automation

Architecture

  • SecKey.Core: config, logging, JSON, utility layer
  • SecKey.Graph: Graph services, token provider, bulk importers
  • SecKey.App: WPF operational shell and workflows
  • SecKey.Cli: command-line automation entrypoint

Security Model

  • Delegated interactive/device-code sign-in
  • Application credentials for unattended operation
  • Scope-driven Graph permissions with admin consent support
  • Tenant-safe object existence checks before create/update
